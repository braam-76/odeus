# Odeus programming language

Odeus - LISP language. Implementation in progress.

## Getting started

Clone the repo, then in clonned folder:

``` sh
mkdir build && cd build
cmake ../
make # or cmake --build .
```

Run REPL:

``` sh
./build/odeus
# or if you in build folder
./odeus
```

Execute file:
``` sh
./build/odeus <filename>
# or if you in build folder
./odeus <filename>
```
